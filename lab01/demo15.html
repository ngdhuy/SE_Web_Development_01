URL = protocol://domain:port/folder/files
  1. protocol://domain -> load default index.html/php/aspx in root (can be config from web server)
  2. protocol://domain/file -> load default root/file
  3. protocol://domain/folder -> load default root/folder/index.html
  4. protocol://domain/folder/file -> load default root/folder/file
  5. protocol://domain?a=2&b=4 -> pass variables a = 2 and b = 4 (query string)
  6. protocol://domain#Name -> move to #Name in destination page
  7. protocol://domain:port -> access with port
