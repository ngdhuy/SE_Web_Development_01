### 1. HTML + CSS

XML Structure

```xml
<root>
  <parents>
    <child></child>
    Data = text
  </parents>

  <single />

  <tag property="value" pro="value">
    Data = text
  </tag>

  <tag id="unique identify" name="can be overlap name" class="reference to class name in CSS" style="define style of CSS here" />

</root>
```

HTML structure

```xml
  <html>
    <head>
      <!-- configuration for webpage -->
    </head>
    <body>
      <!-- content of webpage -->
    </body>
  </html>
```

**Tag type**
1. Text
    - Heading: h1, h2, h3, h4, h5, h6
    - text: span, i, b, u, del
    - link: a
    - paragraph: p
2. Block
    - table
      - tr : table row
      - td : table data
      - th : table head
    - advance of table
      - caption
      - thead : table header
      - tbody : table body
      - tfoot : table footer
    - div : div, header, footer, section, nav
    - list
      - order list : ol, li
      - un-order list: ul, li
3. Media: 
    - image
    - sound (html5)
    - video (html5)
